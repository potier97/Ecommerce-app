## Description

Nest back para api de peliculasm en la que  progresivamente se va agregando distintas funcionalidades.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Pruebas Unitarias

```bash
# unit tests
$ yarn run test
```

Para correr solo una prueba unitaria en particular, se debe correr el siguiente comando:

```bash
# unit tests
$ jest --watch src/nombre-del-archivo.spec.ts
```
